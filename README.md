# Tubes_PMC
Tempat nyatuin kode tubes


## Cara kontribusi:
### 1. Clone Repo:
```
git clone https://github.com/Ario-sut/Tubes_PMC.git
```

### 2. Buat branch baru
```
git checkout -b "nama-branch"
```
Misal "nama-branch" : "branch-bambank"

### 3. Add, commit, push
```
git add .
```
```
git commit -m "message"
```
Misal "message" : update
